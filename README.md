## 규칙
1. 매주 **수요일 11시**에 **온라인으로 미팅**을 갖습니다
2. **공동으로 주제를 선정**한 후 **각자 주제에 해당하는 세션을 1개씩 선택해 공유**합니다
    - 세션 총 분량은 100분 내외
3. 스터디 당일 **랜덤으로 선정된 1명이 정리한 내용을 발표**합니다
4. 예상 소요 시간은 **1시간(QnA 30분, 발표 30분) 내외**입니다.
5. 문제 풀이 내용을 **PR**로 올림
6. 수요일 스터디 이후 PR 머지
7. 벌금
   - 수요일 11시 이전까지 PR올리지 않을시, **벌금 10,000원**
